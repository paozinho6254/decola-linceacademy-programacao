# 6ª interação com IA (Ferramenta utilizada: ChatGPT)

Pergunta: Em quais situações vale mais a pena usar switch em vez de vários if else?
Resposta:
O switch é mais indicado quando você precisa comparar uma mesma variável com vários valores possíveis. Ele deixa o código mais organizado e fácil de ler do que muitos if else seguidos. Já o if else funciona melhor para condições mais complexas, envolvendo comparações diferentes ou operadores lógicos.

Pergunta:Por que é importante ter cuidado com loops while?
Resposta:
O principal cuidado é evitar loops infinitos. Isso acontece quando a condição de parada nunca se torna falsa. Por isso, é importante atualizar corretamente a variável de controle dentro do while, garantindo que o laço termine no momento esperado.

Perguta: Qual a função dos comandos break e continue em um loop?
Resposta:
O break encerra o loop imediatamente, mesmo que ainda existam repetições restantes. Já o continue pula apenas a execução atual e segue para a próxima repetição do laço. Ambos ajudam no controle do fluxo do programa e tornam certas lógicas mais eficientes.
